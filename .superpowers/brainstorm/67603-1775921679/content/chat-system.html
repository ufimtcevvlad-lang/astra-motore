Как работает

💬 Чат на сайте

Виджет в правом нижнем углу. Клиент пишет → админ видит в реальном времени и отвечает из админки. Когда все админы офлайн — виджет становится формой «Оставьте сообщение».

✈️ Telegram

Бот магазина получает сообщения от клиентов → они появляются в едином окне. Админ отвечает из админки → клиент получает ответ в Telegram.

✉️ Почта

Письма на адрес магазина подтягиваются в единое окно (IMAP). Админ отвечает из админки → клиент получает email.

Уведомления

Каждое новое сообщение из любого канала дублируется в Telegram-бота, чтобы админ не пропустил обращение, даже если не сидит в админке. Бейдж в меню показывает количество неотвеченных.
