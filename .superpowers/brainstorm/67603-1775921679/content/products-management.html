Возможности

Список товаров: поиск, фильтры по категории и бренду, пагинация, чекбоксы для массовых действий

Массовые действия: выбрать несколько товаров → изменить цену, остаток или удалить

Импорт Excel: загрузить файл → предпросмотр данных → подтвердить → товары добавлены/обновлены

Экспорт Excel: скачать все товары (наименование, артикул, бренд, цена, остаток) одной кнопкой

Редактирование: все поля товара, drag & drop фото, таблица характеристик, поиск аналогов

Подсветка проблем: товары без фото выделены жёлтым, нулевой остаток — красным
