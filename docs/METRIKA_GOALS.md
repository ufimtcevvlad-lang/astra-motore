# Цели Яндекс.Метрики

Эти цели сайт отправляет сам через `ym(..., "reachGoal", ...)`.
В Метрике нужно завести цели типа **JavaScript-событие** с такими идентификаторами:

| Цель | Идентификатор | Что означает |
| --- | --- | --- |
| Клик по телефону | `phone_click` | Нажали на номер телефона |
| Клик WhatsApp | `whatsapp_click` | Перешли в WhatsApp с сайта |
| Клик Telegram | `telegram_click` | Перешли в Telegram с сайта |
| Добавление в корзину | `add_to_cart` | Товар добавили в корзину |
| Заказ отправлен | `order_sent` | Форма заказа успешно ушла |
| VIN-запрос отправлен | `vin_request_sent` | Форма VIN-запроса успешно ушла |
| Поиск по сайту | `site_search` | Пользователь искал товар по сайту |
| Открытие корзины | `cart_open` | Пользователь открыл корзину |

Главные цели для продаж: `order_sent`, `add_to_cart`, `whatsapp_click`, `phone_click`.
