## Шапка сайта (Excalidraw)

Верхняя строка: **логотип** → поле **«введите номер или название детали»** + кнопка **«Поиск»** → **Корзина** → блок **Войти / Регистрация** (или ЛК).

Ниже: **Главная**, **Каталог** (выпадающий: Витрина, Opel, Chevrolet, GM), **Клиентам** (выпадающий: о компании, как заказать, ПДн, договор, гарантия, возврат), **Оплата и доставка**, **Контакты**, **VIN запрос**. Отдельная полоса «Каталоги» в шапке убрана — марки доступны из меню **Каталог**.

Поиск в шапке ведёт на `/catalog?q=…` (синхронизируется с фильтром каталога).

## Что сделано в коде (`ProductCatalog` + `CatalogGroupNav`)

1. **Шапка сайта** — верхняя строка с поиском и меню; переходы Витрина / Opel / Chevrolet / GM — в выпадающем **Каталог**.
2. **Страница каталога** — `CatalogChrome`: крошки → H1 → текст.
3. **Блок фильтров** (сверху вниз):
   - **Марка** — сегменты «Все / Opel / Chevrolet».
   - **Поиск** — одно поле на всю ширину.
   - **К группам** — горизонтальные якоря на крупные группы (только при «Все разделы» и без текста в поиске); на экранах **&lt; lg** видно в карточке фильтров.
   - **Раздел** — выпадающий список всех подразделов.
   - **Сбросить фильтры** — при активных условиях.
4. **Витрина по группам** — на **lg+** слева липкая колонка «Группы» с теми же якорями; справа — блоки H2 по группам и H3 по разделам с сеткой карточек.

При выборе конкретного раздела или вводе поиска показывается плоская сетка без боковой навигации по группам.
