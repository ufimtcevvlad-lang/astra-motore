# Дизайн: Мониторинг цен конкурентов

**Дата:** 2026-04-13  
**Проект:** GM Shop 66 (gmshop66.ru)  
**Статус:** Утверждён

---

## Цель

Инструмент для мониторинга цен конкурентов по каждой запчасти. Показывает рекомендованную цену на основе анализа рынка. Цену на сайте не меняет автоматически — только рекомендация.

---

│       ПАРСЕР VPS            │     │        ОСНОВНОЙ VPS          │
│                             │     │                              │

Парсер полностью изолирован от основного сайта — отдельный VPS, отдельный IP. Это защищает основной сайт от блокировок.

## Идентификация товара

Поиск у конкурентов — по **артикулу + бренд** (например `95227052` + `ACDelco`). Это самый точный способ — исключает совпадения разных производителей.

Список артикулов хранится в файле `articles.txt` на парсер VPS. Экспортируется из 1С при необходимости.

## Парсер — логика работы

**Ночной режим (автоматический):**
- Запуск: каждую ночь в 02:00 по расписанию (cron)
- Параллельность: 10 потоков одновременно
- Задержка между запросами: 3-10 сек (защита от банов)
- Время на 7000 позиций: ~1-2 часа
- Результат готов к утру

**Ручной режим (по кнопке в админке):**
- Кнопка "Спарсить цену" на карточке товара
- Парсит 5 сайтов прямо сейчас для одного артикула
- Результат появляется через 5-15 секунд
- Обновляет данные в SQLite

**Защита от блокировок:**
- Задержки 3-10 сек между запросами
- Ротация User-Agent
- Отдельный IP (парсер VPS ≠ основной VPS)
- При необходимости в будущем: добавить 2-3 дешёвых VPS (~150 руб/мес каждый) с ротацией IP

## База данных (SQLite)

Один файл `prices.db` на парсер VPS.

**Таблица `prices` — сырые данные по каждому сайту:**
```sql
article    TEXT     -- артикул, например "95227052"
brand      TEXT     -- бренд, например "ACDelco"
site       TEXT     -- источник, например "exist.ru"
price      REAL     -- цена в рублях
updated_at DATETIME -- когда собрано
```

**Таблица `price_summary` — итоговая аналитика:**
```sql
article     TEXT
brand       TEXT
avg_price   REAL     -- среднее арифметическое по всем сайтам
min_price   REAL     -- минимум
max_price   REAL     -- максимум
sites_count INTEGER  -- сколько сайтов нашли этот артикул
updated_at  DATETIME
```

Цена рынка = `avg_price` (среднее арифметическое по найденным сайтам).

## FastAPI — эндпоинты

**GET /price**
```
Параметры: article, brand
Ответ: { avg_price, min_price, max_price, sites_count, updated_at, sites: [...] }
```

**POST /parse**
```
Параметры: article, brand
Действие: немедленный парсинг одного товара
Ответ: обновлённые данные (через 5-15 сек)
```

API закрыт токеном — только основной VPS может к нему обращаться.

## Шкала в админ-панели

Отображается на карточке каждого товара в разделе Admin Panel.

```
Ваша цена: 3 800 руб.    Рынок: ~3 210 руб. (5 магазинов)

[████████████████░░░░░░░░░░░░░░]
 дёшево        норма        дорого

Статус: ⛔ Выше рынка на 18% — рекомендуем снизить до ~3 200 руб.

                              [Спарсить цену]
```

**Логика цвета:**

| Отклонение от avg_price | Цвет | Смысл |
|------------------------|------|-------|
| Ваша цена < avg × 0.85 | 🟡 Жёлтый | Вы дешевле рынка — возможно недополучаете прибыль |
| avg × 0.85 ≤ цена ≤ avg × 1.15 | 🟢 Зелёный | Рыночная цена — всё ок |
| Ваша цена > avg × 1.15 | 🔴 Красный | Выше рынка — риск потерять клиентов |

Пороги ±15% можно изменить в конфиге без переписывания кода.

**Если данных нет** (артикул не найден ни на одном сайте): показывается нейтральный статус "Нет данных".

## Уведомления в админке

После каждого ночного парсинга система формирует уведомления по товарам у которых цена вышла за пределы зелёной зоны.

**Уведомления только для:**
- 🔴 Красный — ваша цена выше рынка на 15%+
- 🟡 Жёлтый — ваша цена ниже рынка на 15%+
- 🟢 Зелёный — не уведомляем

**Колокольчик** (иконка в шапке админки):
- Значок с цифрой — сколько товаров требуют внимания
- Клик → список уведомлений: название товара, артикул, ваша цена, цена рынка, отклонение %

**Утренний дашборд** (главная страница админки):
```
📊 Ночной отчёт — 13 апреля 2026

Обновлено товаров: 6 842 из 7 000
⛔ Выше рынка (🔴):  124 товара
⚠️ Ниже рынка (🟡):   87 товаров
✅ В норме (🟢):    6 631 товар

[Посмотреть список →]
```

Список сортируется по размеру отклонения — самые критичные сверху.

## Что система НЕ делает

- Не меняет цены на сайте автоматически
- Не блокирует продажи при любом статусе шкалы
- Не показывается покупателям — только в Admin Panel
