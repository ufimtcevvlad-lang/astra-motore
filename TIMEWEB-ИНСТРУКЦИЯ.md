# Перенос сайта Astra Motors на Timeweb (русские сервера)

Пошаговая инструкция. Выполняй по порядку.

---

## Шаг 1. Заказать VPS на Timeweb

1. Зайди на **timeweb.com** → войди в личный кабинет.
2. Меню → **VPS** (виртуальные серверы) → **Заказать**.
3. Выбери:
   - **ОС:** Ubuntu 22.04
   - **Тариф:** от 1 ГБ RAM (минимум для Next.js)
4. Задай **пароль root**, сохрани его. Заверши заказ.
5. В панели сервера скопируй **IP-адрес** (например `123.45.67.89`) — понадобится дальше.

---

## Шаг 2. Подключиться к серверу по SSH

На своём компьютере открой **Терминал** (Mac) или PuTTY (Windows) и выполни:

```bash
ssh root@IP_СЕРВЕРА
```

Пример: `ssh root@123.45.67.89`  
Введи пароль root. После входа ты в консоли сервера — все следующие команды вводи здесь.

---

## Шаг 3. Установить Node.js, Git и PM2

На сервере выполни по очереди:

```bash
apt update && apt upgrade -y
```

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install -y nodejs
```

```bash
apt install -y git
npm install -g pm2
```

Проверь: `node -v` (должно быть v20.x.x).

---

## Шаг 4. Скачать проект с GitHub на сервер

На сервере:

```bash
mkdir -p /var/www
cd /var/www
git clone https://github.com/ufimtcevvlad-lang/astra-motore.git astra-motors
cd astra-motors
```

**Если репозиторий приватный:** в GitHub создай Personal Access Token (Settings → Developer settings → Personal access tokens, право **repo**). Тогда команда такая:

```bash
git clone https://ТВОЙ_ТОКЕН@github.com/ufimtcevvlad-lang/astra-motore.git astra-motors
cd astra-motors
```

---

## Шаг 6. Собрать и запустить сайт

На сервере, в папке `/var/www/astra-motors`:

```bash
npm install
npm run build
```

Запуск через PM2:

```bash
NODE_ENV=production pm2 start npm --name "astra-motors" -- start
pm2 save
pm2 startup
```

На вопрос команды `pm2 startup` выбери вариант для своего пользователя (обычно копируешь и вставляешь предложенную команду).  
Проверка: `curl http://localhost:3000` — должна вернуться страница в HTML.

---

## Шаг 7. Установить Nginx (веб-сервер снаружи)

На сервере:

```bash
apt install -y nginx
nano /etc/nginx/sites-available/astramotors
```

В открывшийся файл вставь (если домен другой — замени **astramotors.shop**):

```nginx
server {
    listen 80;
    server_name astramotors.shop www.astramotors.shop;
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

Сохрани (**Ctrl+O**, Enter) и выйди (**Ctrl+X**). Затем:

```bash
ln -sf /etc/nginx/sites-available/astramotors /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default
nginx -t
systemctl reload nginx
```

Проверка: в браузере открой **http://IP_СЕРВЕРА** (твой IP с шага 1). Должен открыться сайт (пока без HTTPS).

---

## Шаг 8. Включить HTTPS (SSL)

На сервере:

```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d astramotors.shop -d www.astramotors.shop
```

Введи email, согласись с условиями. Certbot сам настроит Nginx на HTTPS.  
После этого сайт будет открываться по **https://astramotors.shop**.

---

## Шаг 9. Настроить DNS домена

Чтобы сайт открывался по адресу **astramotors.shop** (а не по IP), нужно направить домен на твой VPS. Для этого в панели, где куплен домен, добавляются DNS-записи.

**IP твоего сервера** (из шага 1) — подставь его везде, где ниже написано «IP_СЕРВЕРА». Пример: `5.42.117.221`.

---

### Если домен куплен на Timeweb

1. Зайди на [timeweb.com](https://timeweb.com) → войди в личный кабинет.
2. В меню слева выбери **Домены** (или **Мои домены**).
3. Найди домен **astramotors.shop** и нажми на него (или кнопку «Управление» / «Настроить»).
4. Открой раздел **DNS-записи** / **Управление зоной** / **Редактировать зону**.
5. **Запись для основного домена (без www):**
   - Нажми **Добавить запись** (или «Добавить»).
   - **Тип записи:** A.
   - **Имя (хост, поддомен):** оставь пустым или введи **@** (это значит «сам домен astramotors.shop»).
   - **Значение / IP-адрес:** введи IP твоего VPS, например `5.42.117.221`.
   - **TTL:** можно оставить по умолчанию (300 или 3600). Сохрани запись.
6. **Запись для www (чтобы открывался и www.astramotors.shop):**
   - Снова **Добавить запись**.
   - **Тип:** A.
   - **Имя:** `www`.
   - **Значение:** тот же IP сервера (например `5.42.117.221`). Сохрани.
7. Убедись, что старых записей типа A для **@** или **www** с другими IP нет (если есть — удали или замени значение на IP своего VPS).

---

### Если домен на Reg.ru

1. Зайди на [reg.ru](https://www.reg.ru) и войди в **Личный кабинет** (правый верхний угол).
2. В верхнем меню выбери **Домены** (или в боковом меню — раздел «Домены»).
3. В списке найди домен **astramotors.shop** и нажми на него (или на кнопку **«Управление»** рядом с ним).
4. Откроется страница домена. Найди блок **«DNS-серверы и управление зоной»** и нажми **«Изменить»** или **«Подробнее»** (иногда ссылка называется **«Управление зоной»**).
5. Попадёшь в **редактор зоны DNS**. Там таблица с записями (A, AAAA, CNAME, MX и т.д.). Нажми **«Добавить запись»** (или «Добавить ресурсную запись»).
6. **Первая запись (сайт без www):**
   - Тип: **A**.
   - Имя / Субдомен: оставь **пустым** или введи **@** (это «корень» домена — сам astramotors.shop).
   - Значение / IP: введи IP твоего VPS (например `5.42.117.221`). Сохрани.
7. **Вторая запись (www.astramotors.shop):** снова **«Добавить запись»** → тип **A**, имя **www**, значение — **тот же IP**. Сохрани.
8. Если уже есть старые A-записи для **@** или **www** с другим IP — удали их или измени значение на IP твоего сервера.

---

### Если домен на другом регистраторе (Nic.ru, Beget и т.д.)

Общий принцип тот же:

- Найти раздел **DNS**, **Управление зоной**, **Редактировать записи** для домена **astramotors.shop**.
- Добавить **A-запись** для корня домена (хост **@** или пусто): значение — **IP твоего VPS**.
- Добавить **A-запись** для **www**: значение — **тот же IP**.

В некоторых панелях «корень» обозначается как «@» или пустое поле «Имя».

---

### После сохранения записей

- Изменения DNS применяются не сразу: обычно **5–30 минут**, иногда до 1–2 часов.
- Проверка: открой в браузере **https://astramotors.shop** (и при желании **https://www.astramotors.shop**). Должен открываться твой сайт с сервера Timeweb.
- Если открывается старая версия или ошибка — подожди ещё или проверь, что в DNS указан именно IP твоего VPS и нет опечаток.

---

## Шаг 10. Отключить домен от Vercel (чтобы работал только Timeweb)

Чтобы трафик шёл только на Timeweb:

1. Зайди на **vercel.com** → проект **astra-motors**.
2. **Settings** → **Domains** → найди **astramotors.shop** → **Remove** (удалить домен из Vercel).
3. DNS домена уже указывает на IP Timeweb (шаг 9) — ничего больше менять не нужно.

Сайт будет работать только с серверов Timeweb в РФ.

---

## Как обновлять сайт после изменений в коде

Когда что-то меняешь в проекте и пушишь в GitHub, на сервере выполни:

```bash
cd /var/www/astra-motors
git pull origin main
npm install
npm run build
pm2 restart astra-motors
```

Сайт обновится за 1–2 минуты.
