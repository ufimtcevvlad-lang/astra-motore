# Фото каталога (пилот Opel)

Изображения с **Wikimedia Commons** — иллюстрация **типа** детали, не гарантия совпадения с конкретным OEM на фото. Лицензии: страницы файлов на Commons.

Повторная загрузка всех превью:

```bash
npm run catalog:images
```

(скрипт `scripts/download-catalog-images-curated.mjs` — один URL качается один раз, копии на связанные позиции.)
